- **Kafka-less**: Прямая отправка в Pub/Sub

<div className="glass-card p-4 rounded-lg">

### Google Pub/Sub
- **Topics**: Аналог Kafka topics
- **Subscriptions**: Push и Pull модели
- **Ordering keys**: Гарантия порядка

</div>

<div className="glass-card p-4 rounded-lg">

- **File-based**: Локальный файл

- **Redis**: Для HA deployments

</div>

<div className="glass-card p-4 rounded-lg">

### Cloud Run
- **Serverless containers**: Автоскейлинг
- **Workload Identity**: Безопасная аутентификация
- **VPC connector**: Private network access

</div>

## Kafka vs Debezium Server

| Аспект | Kafka Connect | Debezium Server |
|--------|---------------|-----------------|
| Зависимости | Kafka обязателен | Kafka не нужен |
| Масштабирование | Kafka partitions | Вертикальное |
| Offset storage | Kafka topics | Файл/Redis/GCS |
| Sink flexibility | Только Kafka | Pub/Sub, Kinesis, HTTP |
