## Ключевые концепции

<div className="grid gap-4 md:grid-cols-2 mb-8">

<div className="glass-card p-4 rounded-lg">

### Multi-Database Architecture
- **PostgreSQL + MySQL**: Единый pipeline
- **Unified vs Separate topics**: Выбор паттерна
- **Schema management**: Разные механизмы

</div>

<div className="glass-card p-4 rounded-lg">

### Production Architecture
- **HA Kafka Connect**: Distributed mode
- **Monitoring stack**: Prometheus + Grafana
- **Alerting**: PagerDuty/Slack integration

</div>

<div className="glass-card p-4 rounded-lg">

### Operational Differences
- **PostgreSQL**: Slots, LSN, WAL bloat
- **MySQL**: GTID, binlog retention, schema history
- **Common**: Lag monitoring, snapshot handling

</div>

<div className="glass-card p-4 rounded-lg">

### Deliverables
- **Architecture diagram**: Компоненты и потоки
- **Monitoring dashboard**: Ключевые метрики
- **Runbook**: Операционные процедуры

</div>
