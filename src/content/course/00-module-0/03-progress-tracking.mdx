# Отслеживание прогресса

Курс автоматически отслеживает ваш прогресс. Вот как это работает.

---

## Как отметить урок пройденным

В конце каждого урока есть **чекбокс "Отметить как пройденный"**.

**Что происходит при отметке:**

1. Урок добавляется в ваш список пройденных
2. В боковом меню появляется ✓ рядом с названием урока
3. Счётчик прогресса модуля обновляется (например, `3/6` → `4/6`)

Вы можете снять отметку, если хотите пройти урок заново.

## Где хранится прогресс

Прогресс сохраняется в **localStorage вашего браузера**.

**Что это значит:**

| Аспект | Описание |
|--------|----------|
| **Локально** | Данные хранятся только в вашем браузере |
| **Без сервера** | Ничего не отправляется в интернет |
| **Привязка к браузеру** | Другой браузер = отдельный прогресс |
| **Приватность** | Ваши данные остаются у вас |

---

## Когда прогресс может сброситься

Прогресс будет потерян если:

- **Очистите данные браузера** — localStorage удаляется вместе с кэшем
- **Используете режим инкогнито** — данные не сохраняются между сессиями
- **Смените браузер** — у каждого браузера свой localStorage

**Рекомендация:** Если хотите сохранить прогресс надолго, используйте один браузер и не очищайте данные сайтов.

---

## Просмотр общего прогресса

Ваш прогресс виден в нескольких местах:

1. **Боковое меню** — счётчик `X/Y` рядом с каждым модулем
2. **Главная страница** — общий индикатор прогресса по курсу
3. **Список уроков** — галочки ✓ у пройденных уроков
