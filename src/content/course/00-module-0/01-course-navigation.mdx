# Навигация по курсу

Добро пожаловать в курс по Debezium! Этот урок поможет вам быстро освоиться с платформой и начать обучение.

---

## Боковое меню (Sidebar)

Слева на экране находится **боковое меню** — ваш главный навигатор по курсу.

**Что вы найдёте в меню:**

- **Модули** — основные разделы курса (от 0 до 8)
- **Уроки** — темы внутри каждого модуля
- **Прогресс** — сколько уроков вы уже прошли в каждом модуле

**Как пользоваться:**

1. Кликните на **название модуля**, чтобы развернуть список уроков
2. Кликните на **урок**, чтобы перейти к нему
3. Счётчик справа от модуля показывает прогресс (например, `3/6`)

На мобильных устройствах меню скрыто — используйте иконку гамбургера (☰) в верхнем левом углу.

---

## Главная страница

На главной странице курса вы найдёте:

- **Accordion-меню модулей** — краткий обзор всех модулей с описанием
- **Прогресс по курсу** — визуальный индикатор вашего продвижения
- **Быстрый доступ** — к последнему уроку или следующему непройденному

Чтобы вернуться на главную, кликните на **логотип курса** в шапке сайта.

---

## Структура урока

Каждый урок содержит:

| Элемент | Описание |
|---------|----------|
| **Заголовок** | Название темы |
| **Время** | Примерное время на изучение |
| **Сложность** | beginner / intermediate / advanced |
| **Теория** | Объяснение концепций с примерами |
| **Код** | Примеры конфигураций и скриптов |
| **Диаграммы** | Интерактивные схемы архитектуры |

---

## Навигация между уроками

В конце каждого урока есть кнопки:

- **← Предыдущий урок** — вернуться назад
- **Следующий урок →** — перейти к следующей теме

Также вы можете использовать боковое меню для перехода к любому уроку.

---

## Поиск по курсу

Для быстрого поиска по всем материалам курса используйте **глобальный поиск**.

**Как открыть поиск:**

- **macOS**: нажмите `⌘ + K` (Cmd + K)
- **Windows/Linux**: нажмите `Ctrl + K`

**Что можно искать:**

- Названия уроков и модулей
- Ключевые слова в тексте
- Фрагменты кода и конфигураций
- Термины и концепции

**Управление поиском:**

| Клавиша | Действие |
|---------|----------|
| `↑` / `↓` | Навигация по результатам |
| `Enter` | Перейти к выбранному результату |
| `Escape` | Закрыть поиск |

Результаты поиска показывают **название урока** и **фрагмент текста** с подсвеченными совпадениями.

## Ключевые выводы

1. **Боковое меню** — основной способ навигации по курсу
2. **Клик на модуль** разворачивает список уроков
3. **Главная страница** даёт обзор всего курса
4. Используйте **кнопки навигации** внизу уроков для последовательного прохождения
5. **Cmd+K / Ctrl+K** открывает быстрый поиск по всем материалам
