# Структура курса

Курс состоит из 8 модулей, которые проведут вас от основ до production-ready CDC-пайплайнов.

---

## Обзор модулей

| # | Модуль | Что изучите | Сложность |
|---|--------|-------------|-----------|
| 0 | **Как пользоваться** | Навигация, прогресс, подготовка | Beginner |
| 1 | **Введение в CDC** | Основы CDC, первый коннектор, структура событий | Beginner |
| 2 | **PostgreSQL и Aurora** | WAL, replication slots, Aurora PostgreSQL | Intermediate |
| 3 | **MySQL/Aurora** | Binlog, GTID, Aurora MySQL | Intermediate |
| 4 | **Prod Operations** | Мониторинг, JMX метрики, troubleshooting | Intermediate |
| 5 | **SMT и Паттерны** | Трансформации, Outbox pattern, routing | Advanced |
| 6 | **Data Engineering** | Python, PyFlink, обработка CDC-событий | Advanced |
| 7 | **Cloud-Native GCP** | Debezium Server, Pub/Sub, Cloud Run | Advanced |
| 8 | **Capstone Project** | Полный production pipeline | Advanced |

---

## Рекомендуемый путь

### Для начинающих с CDC

```
Module 0 → Module 1 → Module 2 → Module 4 → Module 5
```

Сначала основы, потом PostgreSQL (самый популярный), затем операции и паттерны.

### Для работающих с MySQL

```
Module 0 → Module 1 → Module 3 → Module 4 → Module 5
```

Замените Module 2 на Module 3 для фокуса на MySQL/Aurora MySQL.

### Для дата-инженеров

```
Module 0 → Module 1 → Module 6 → Module 5
```

Быстро к Python-интеграции и обработке событий.

### Полный курс

```
Module 0 → 1 → 2 → 3 → 4 → 5 → 6 → 7 → 8
```

Последовательное прохождение всех модулей для максимального охвата.

---

## Пререквизиты

Курс предполагает, что вы уже знакомы с:

- **SQL** — базовые запросы, понимание транзакций
- **Docker** — запуск контейнеров, docker compose
- **Python** — для модулей 5-6 (опционально)
- **Kafka** — базовое понимание topics/consumers (объясним по ходу)

Не знаете Kafka? Модуль 1 даст достаточно контекста для старта.
